# ReiParticleSkill Forge Module

Minecraft Forge 1.20.1 particle skill mod with custom Ender Dragon respawn animations and visual effects.

The `ReiParticlesAPI` runtime is now bundled directly into this module. Building `forge-port` produces the only jar you need.

## Features

- Custom particle animations for the Ender Dragon respawn ritual
- Integrated particle runtime, display system, and command ports
- Custom projectile entities and gameplay-driven effects

## Building

```bash
.\gradlew build
```

Output: `build/libs/reiparticleskill-1.0-SNAPSHOT-forge-port.jar`

## Development

```bash
.\gradlew runClient
```
