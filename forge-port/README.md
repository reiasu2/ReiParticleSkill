# ReiParticleSkill Forge Module

Minecraft Forge 1.20.1 particle skill mod with custom Ender Dragon respawn animations and visual effects.

## Features

- Custom particle animations for the Ender Dragon respawn ritual (shockwave, rotating magic circles, beam replacement)
- Particle skill system
- Custom projectile entities

## Building

```bash
.\gradlew build
```

Output: `build/libs/reiparticleskill-1.0-SNAPSHOT-forge-port.jar`

## Development

```bash
.\gradlew runClient
```

## Dependencies

Requires the `forge-port-api` module jar (ReiParticlesAPI) at runtime.
